# 📝 Task Manager (Laravel)

A web-based Task Manager application built using Laravel to manage daily tasks efficiently with full CRUD functionality.

---

## 🚀 Features
- ➕ Add new tasks  
- 📋 View all tasks  
- ✏️ Edit existing tasks  
- ❌ Delete tasks  
- ✅ Form validation  
- 🎨 Clean UI using Bootstrap  

---

## 🛠 Tech Stack
- Laravel  
- PHP  
- MySQL  
- Bootstrap  

---

## 📂 Project Structure
- MVC Architecture (Model-View-Controller)  
- Blade templates for frontend  
- Eloquent ORM for database handling  

---
